# Template Selection

Expo Router mobile starter for React Native apps that run in Expo Go.

Use when:
- The user asks for a mobile, iOS, Android, phone, React Native, or Expo app
- The request is primarily local UI, navigation, device-friendly layout, or prototype behavior
- The app can work without backend persistence, realtime shared state, auth, uploads, or server-side data
- The user needs Expo Go preview compatibility with no native build setup

Avoid when:
- The request needs persistence, auth, realtime collaboration, uploads, shared data, or backend functions
- The user asks for a web-only app, Cloudflare Worker app, presentation, or backend workflow
- The request requires custom native modules, dev clients, EAS builds, prebuild, or app-store submission

Built with:
- Expo Router, React Native, Expo Go-compatible packages, TypeScript, Expo web preview
